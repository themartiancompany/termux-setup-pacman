=====================
termux-setup-pacman
=====================

--------------------------------------------------------------
Sets up a pacman environment in Termux
--------------------------------------------------------------
:Version: termux-setup-pacman |version|
:Manual section: 1

Synopsis
========

termux-setup-pacman *[option]* *action*

Description
===========

Sets up a pacman package manager based
Termux environment.

Actions
=========

* prepare
* install
* clean
* setup

Options
=======

-h                      Display help.
-c                      Enable color output
-v                      Enable verbose output
